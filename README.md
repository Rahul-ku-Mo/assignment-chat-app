# Step to run the project in local

### 1. git clone url
### 2. cd .frontend (use tab key) 
### 3. npm install
### 4. npm run dev
### 5. cd..
### 6. cd .backend (use tab key)
### 7. npm install
### 8. nodemon app.js

Now you are good to go! 

# Frontend
## To run in local use npm run dev

# Backend
## To run in local use nodemon app.js
